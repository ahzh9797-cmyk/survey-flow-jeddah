## النشر على Vercel (الطريقة الموصى بها)

### الخطوة 1: ارفع هذا المجلد إلى GitHub
1. أنشئ مستودع (repository) جديد على GitHub
2. ارفع كل ملفات هذا المجلد إليه

### الخطوة 2: اربطه بـ Vercel
1. اذهب إلى [vercel.com](https://vercel.com) وسجّل دخول (يمكنك تسجيل الدخول بحساب GitHub مباشرة)
2. اضغط **"Add New" → "Project"**
3. اختر المستودع الذي رفعته للتو
4. Vercel سيكتشف تلقائياً أنه مشروع Vite — لا تغيير أي إعدادات
5. اضغط **"Deploy"**

بعد دقيقة أو دقيقتين، سيكون موقعك جاهزاً بدومين مجاني مثل:
`survey-flow-jeddah.vercel.app`

### الخطوة 3 (اختياري): دومين مخصص
من إعدادات المشروع في Vercel → **Domains** يمكنك ربط دومين خاص بك إن توفر لديك.
